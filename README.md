## Featured Projects

### Study Plan Modification Management System

Frontend web application developed as part of my Computer Engineering diploma thesis.

The system supports the management of academic study plans, curriculum structures, modification requests, approval workflows, version consultation and document download.

**Main technologies:** Vue.js, JavaScript, Axios, Laravel API, MySQL.

Repository: [frontend_sgmpe](https://github.com/Sunaymg04/frontend_sgmpe)

---

### Academic Management API

Laravel-based REST API used as part of a collaborative academic management ecosystem.

The backend supports academic entities such as faculties, departments, careers, curricula, subjects and study plans, and provides data used by academic management modules.

**Main technologies:** Laravel, PHP, MySQL, REST API.

Repository: [api-laravel](https://github.com/Sunaymg04/api-laravel)
